## About This Project

This project was created as part of my Open Source Software course. I worked on it using Ubuntu inside a virtual machine (UTM).

## Chosen Software

I chose **Python** for this project.

Python is an open-source programming language that is widely used today in areas like web development, artificial intelligence, and automation. It is known for its simple and readable syntax, which makes it beginner-friendly.

---

## Scripts Overview

### Script 1 — System Identity Report
This script displays basic information about the system such as kernel version, user, uptime, and Linux distribution. It works like a simple system introduction.

---

### Script 2 — FOSS Package Inspector
This script checks whether Python is installed on the system or not. It also prints a short description using a case statement.

---

### Script 3 — Disk and Permission Auditor
This script checks important system directories and shows their permissions and disk usage. It also checks the Python path on the system.

---

### Script 4 — Log File Analyzer
This script reads a log file and counts how many times a keyword (like "error") appears. It also shows the last few matching lines. While working on this, I understood how loops and conditions work in shell scripting.

---

### Script 5 — Open Source Manifesto Generator
This script asks some questions from the user and creates a small manifesto file based on the answers. It was interesting to see how input and file writing works.

---

## How to Run the Scripts

First open terminal and go to the project folder:

```bash
cd OSS_Project
```
Give execute permission to all scripts:
```bash
chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh
```
Run the scripts one by one:
```bash
./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog
./script5.sh
```
